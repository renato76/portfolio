Projects

TMDB Movies Database

Inspired by TMDB and Netflix, this app allows users to view popular movies info and trailers. Once registered users can leave comments and ratings on movies, and also create, update and delete movies. The backend was built using Python, Django REST Framework and PostgreSQL databese, frontend created with React.js and Sass.

PopQuest - Group Project

The popQuest app allows users to create and complete fun 'quests' within the city of their choosing. Each quest consists of a series of stops where the user must solve a clue in order to reveal the next stop. In some cases, the user device's coordinates will 'solve' the clue based on its proximity to a given location. Quests can be timed for those seeking a challenge, or simply a 'guided tour'-like trip.

Formula 1 RESTful API

This app shows the 2020 Season of the Formula One Championship driver details and current standings. The backend was built with Node.js and Express.js, with the database served using MongoDB and Mongoose. The frontend was created using React.js and Bulma with SASS for styling. The app allows full CRUD functionality so users can edit drivers and championship points as the season progresses.

Connect 4 - JS Game

Based on the classic board game where players need to form 4 in a row. This was my first project from General Assembly's Web Development Immersive Course. It was an individual project built in a week, and was both the first proper game I had built, and my first real-world type practice with JavaScript.
